# 4D Results App — Product Definition

## Overview
A fast, clean mobile app for checking Malaysia 4D lottery results:
- Sports Toto
- Magnum
- Da Ma Cai

The app focuses on speed, simplicity, and daily usability.

---

## Target Users
- Malaysian users who check 4D results frequently
- Users who want quick access without ads overload
- Users interested in tracking and future prediction

---

## Core Features (V1)

### 1. Latest Results
- Show latest results for:
  - Toto
  - Magnum
  - Da Ma Cai
- Display:
  - 1st Prize
  - 2nd Prize
  - 3rd Prize
- Fast loading

---

### 2. History
- View past results
- Filter by date
- Simple list view

---

### 3. Notifications
- Notify users when new results are released
- Optional toggle

---

## Future Features (V2)

### 4. Save Numbers
- Users can save favorite numbers
- Track performance

### 5. AI Prediction
- Show hot numbers / trends
- Basic probability insights

---

## Monetization

### Phase 1
- Google AdMob (banner + interstitial)

### Phase 2
- Premium:
  - Remove ads
  - Advanced features (AI insights)

---

## Design Principles
- Minimal UI
- Fast loading
- One-hand usage
- Clear typography for numbers

---

## Success Criteria
- Daily active users
- Fast result access (<2 seconds)
- Retention via notifications
